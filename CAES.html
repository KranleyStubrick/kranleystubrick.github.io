C++ AES Implementation from Scratch

Link to File

This project represents a meticulous and comprehensive implementation of the Advanced Encryption Standard (AES) algorithm entirely from scratch using C++. The primary objective is to delve into the intricate details of cryptographic concepts, with a focus on Galois field mathematical functions, key expansion, and the nuanced AES encryption and decryption processes.

Key Components:

  • Galois Field Mathematics: Implementation of Galois field operations is at the core of this project. This includes the SubBytes transformation, ShiftRows operation, MixColumns operation, and the critical AddRoundKey step. These operations form the foundation of AES and are meticulously crafted to ensure accuracy and efficiency.
  • Key Expansion: The project includes a robust key expansion algorithm that generates round keys from the initial cipher key. This process involves applying various transformations and calculations to create a set of round keys used in the successive encryption rounds.
  • C++ Programming: The entire AES algorithm is implemented in C++, showcasing expertise in low-level programming, algorithmic design, and memory management. The use of C++ provides a balance between efficiency and readability in handling complex cryptographic operations.

Encryption Process Overview:

The AES encryption process involves several well-defined steps:

  1. Key Expansion: The original cipher key is expanded into a set of round keys using the key expansion algorithm.
  2. Initial Round: The input data is combined with the initial round key using the AddRoundKey operation.
  3. Main Rounds: A series of rounds, each comprising SubBytes, ShiftRows, MixColumns, and AddRoundKey operations, are performed to iteratively transform the data.
  4. Final Round: The last round excludes the MixColumns operation, providing the final encrypted output.

This project offers a comprehensive and educational resource for understanding the inner workings of AES, from its mathematical foundations to the practical implementation in C++. It serves as a testament to a commitment to cryptographic excellence, algorithmic precision, and a deep understanding of secure coding practices.
